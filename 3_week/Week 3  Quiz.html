Question 8

Incorrect
Mark 0.00 out of 1.00

Question text

What's the output of the following Ruby script?

r1 = "k".."e"
r1.each {|x| puts x}
Select one:

Feedback

Be very cautious over when dealing with String ranges! When we try to iterate, we start with a value of "k" and enter a loop that terminates when the current value is greater than the right-hand endpoint. But because "k" and "e" are strings, not numbers, they are compared as such. We find that the left endpoint is greater than the right endpoint. So we don't loop at all.
The correct answer is: Prints no output
